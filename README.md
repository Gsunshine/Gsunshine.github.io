# Personal Academic Website

This repository contains the source for my personal academic homepage, built with Jekyll and deployed with GitHub Pages.

## Tech Stack

- Jekyll `3.9.x`
- Ruby + Bundler
- AcademicPages / Minimal Mistakes based theme

## Local Preview

```bash
bundle install
bundle exec jekyll serve --host 127.0.0.1 --port 4000 --force_polling
```

Then open: `http://127.0.0.1:4000`.

## Deployment

Deployment is handled by GitHub Pages from the branch configured in repository settings.
